# SoilGuard AI — Smart Agriculture Drone Platform

A professional, high-performance web platform for real-time drone telemetry, agricultural image analysis, and smart farm management.

## 🚀 Local Development

To run the platform locally, follow these steps:

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Configure Environment Variables:**
   Rename `.env.example` to `.env` and add your Supabase and AI API keys.

3. **Start Development Server:**
   ```bash
   npm run dev
   ```

4. **Access the Platform:**
   Once the server is running, you can access it at:
   👉 **[http://localhost:5173/](http://localhost:5173/)**

## 🛠 Features
- **Real-time Telemetry:** Monitor battery, signal, and altitude.
- **AI Diagnostics:** Analyze soil health and crop status using Gemini AI.
- **Drone Control Hub:** Mission planning and manual flight controls.
- **Cloud Gallery:** Seamless SD card sync and media management.

## 📦 Deployment
This project is optimized for deployment on Vercel, Netlify, or GitHub Pages. Ensure all environment variables listed in `.env.example` are added to your hosting provider's dashboard.
